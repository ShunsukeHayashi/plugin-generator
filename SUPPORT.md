# Support Policy

## Getting Help

### Documentation

First, check the documentation:
* [README.md](README.md) - Getting started guide
* [FAQ](FAQ.md) - Frequently asked questions
* [TROUBLESHOOTING.md](TROUBLESHOOTING.md) - Common issues

### Community Support

For community support:
* **GitHub Issues**: [Create an Issue](https://github.com/ShunsukeHayashi/plugin-generator/issues)
* **GitHub Discussions**: [Join the discussion](https://github.com/ShunsukeHayashi/plugin-generator/discussions)

### Reporting Bugs

When reporting bugs, please include:
1. Plugin Generator version
2. Claude Code version
3. Operating system
4. Steps to reproduce
5. Expected vs actual behavior
6. Error messages or logs

## Support Scope

### Supported

* Installation and setup
* Usage questions
* Bug reports
* Feature requests

### Out of Scope

* Custom plugin development (beyond templates)
* Third-party tool integration
* Claude Code general usage

## Response Time

We aim to respond to:
* **Bug reports**: Within 7 days
* **Feature requests**: Within 14 days
* **Questions**: Within 7 days

Response times may vary based on complexity and maintainer availability.

## Priority Levels

| Priority | Response Time | Examples |
|----------|---------------|----------|
| P0 - Critical | 24-48 hours | Security issues, data loss |
| P1 - High | 3-7 days | Broken core functionality |
| P2 - Medium | 7-14 days | Minor bugs, feature requests |
| P3 - Low | 14+ days | Documentation, enhancements |

## Paid Support

Paid support is not currently available. This is a community-driven project.
