# コントリビューションガイドライン

Plugin Generatorへの貢献を検討いただき、ありがとうございます！🎉

---

## 🤝 はじめに

Plugin GeneratorはClaude Codeのプラグインを素早く作成するためのツールです。このプロジェクトへの貢献を歓迎します。

---

## 📋 貢献の種類

### バグ報告

バグを見つけた場合は、GitHub Issuesで報告してください：

1. 既存のIssuesを検索
2. 新しいIssueを作成
3. バグの再現手順を詳細に記述
4. 環境情報（OS、Claude Codeバージョンなど）を含める

### 機能リクエスト

新しい機能の提案も歓迎します：

1. 既存のIssuesを検索
2. 新しいIssueを作成
3. 機能の概要とユースケースを説明
4. 実装のアイデアがあれば提案

### コードの貢献

直接コードを提出して貢献することも可能です：

1. フォークしてブランチを作成
2. 変更をコミット
3. プルリクエストを作成
4. レビュー待機

---

## 🛠️ 開発環境のセットアップ

### リポジトリのクローン

```bash
git clone https://github.com/ShunsukeHayashi/plugin-generator.git
cd plugin-generator
```

### プラグインのインストール

```bash
claude plugin install ./
```

---

## 📝 コーディング規約

### プラグイン名の命名規則

- 英小文字（a-z）のみ使用
- 数字（0-9）を使用可能
- ハイフン（-）を使用可能
- 先頭は文字で始める
- 3〜50文字以内

### ファイル構成

```
plugin-generator/
├── .claude-plugin/
│   └── plugin.json          # マニフェスト
├── commands/                 # スラッシュコマンド定義
│   └── create-plugin.md
├── templates/                # テンプレートファイル
└── README.md
```

---

## ✅ プルリクエストのチェックリスト

プルリクエストを作成する前に確認してください：

- [ ] 既存のPRと重複していない
- [ ] コードがプロジェクトのスタイルに従っている
- [ ] 変更を説明するコミットメッセージがある
- [ ] ドキュメント（README等）を更新している
- [ ] テストが通っている（またはテストを追加している）
